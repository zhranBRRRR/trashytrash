# PilahYuk

PilahYuk adalah aplikasi yang dapat membantu Anda mengidentifikasi berbagai jenis sampah serta menghitung pengurangan emisi dan nilai barang menggunakan AI Generatif dan sistem RAG yang canggih.  
> Coba di [demonya](https://pilahyuk.vercel.app)

---

PilahYuk is an app that can help you identify different types of trash and calculate reduced emissions and worth of items using Generative AI and advanced RAG systems.  
> Try it in the [demo](https://pilahyuk.vercel.app)
